The Code for APP


                        //Get user's input
                        function getValue() {
                        
                            document.getElementById("alert").classList.add("invisible");
                        
                            let userString = document.getElementById("userString").value;
                        
                            let returnObj = palindromeCheck(userString);
                        
                            displayMessage(returnObj);
                        }
                        
                        //Palindrome check
                        function palindromeCheck(userString) {
                            //To lower case and remove spaces and special characters
                            userString = userString.toLowerCase();
                            let regex = /[^a-z0-9]/gi;
                            userString = userString.replace(regex, "");
                        
                            let revString = [];
                            let returnObj = {};
                        
                            for (let i = userString.length - 1; i >= 0; i--) {
                                revString += userString[i];
                            }
                        
                            if (revString == userString) {
                                returnObj.msg = "Well done, you entered a palindrome!";
                            } else {
                                returnObj.msg = `Sorry, ${userString} is not a palindrome.`;
                            }
                        
                            returnObj.reversed = revString;
                        
                            return returnObj;
                        }
                        
                        //Display message to the user
                        function displayMessage(returnObj) {
                            document.getElementById("alertHeader").innerHTML = returnObj.msg;
                            document.getElementById("message").innerHTML = `Your phrase reversed is ${returnObj.reversed}`;
                            document.getElementById("alert").classList.remove("invisible");
                        }

The code is structured in three functions, getValue(), palindromeCheck() & displayMessage()

getValue()

This function is triggered by the event listener document.getElementById("btnSubmit").addEventListener("click", getValue); when the button Check it! is clicked on the app page. The function first adds the class invisible to the div with the id 'alert'. This ensures that the div is initially hidden from the user. The class is added with the method, document.getElementById("alert").classList.add("invisible"); The function then creates two variables, userString, whose value is passed with the document.getElementById("userString").value; method. This is the user's input, a word or a phrase which the app will determine if it is a palindrome. Next, we have a returnObj variable whose contents will be passed from the palindromeCheck function. This function will return an object, an object enables us to store and retrieve different attributes. For example, we can define a person object and store their firstName & lastName like this: const person = {firstName: Arun, lastName: Pun}; We can then access the value of firstName with the prefix person.firstName which will return Arun.

palindromeCheck()

This function takes in a parameter, userString which we pass as an argument when the getValue() function executes. This string is then converted to lowercase with the String.toLowerCase() method. Next, we have a regular expression, regex to clear up the string further, this variable is defined as /[^a-z0-9]/gi. I then used the String.replace(regex, "") method which replaces any numbers, special characters with blanks, ("") and ensures that there are no invalid characters. Then, we have a reversedString array which will hold the reversed user's input and an empty object defined as returnObj which will hold the appropriate message and the reversed string to display to the user. In a decremental for loop, we start to loop from the last index of the user's input until the loop reaches the first index, 0, decrementing the value of i by 1 on each loop (i--). This loop stores the user's input in reverse in the reversedString variable. Now, we can check if the userString is the same as the reversedString, if the boolean expression evaluates to true, the message Well done, you entered a palindrome! is stored as a msg attribute in the returnObj object. Else, the message, `Sorry, ${userString} is not a palindrome` is stored in the msg attribute. Further to this, the reversed string itself is stored as reversed attribute in the returnObj object no matter the outcome of the boolean expression. Finally, this function returns the returnObj object.

displayMessage()

Now that we have the result, i.e. if the user's input is a palindrome or not, we can simply access the msg attribute in the returnObj object and pass that to the corresponding display element. In this case, the message is to be displayed in a h4 element with the id of alertHeader. Underneath this h4 tag, we have a p element with the id message which will display the user's input in reverse. These are achieved through similar methods, document.getElementById("alertHeader").innerHTML = returnObj.msg; & document.getElementById("message").innerHTML = `Your phrase reversed is ${returnObj.reversed}`; respectively. Finally, in order to display these messages to the user, we need to remove the class invisible from the div element with the id alert. This is done with the document.getElementById("alert").classList.remove("invisible"); method.
